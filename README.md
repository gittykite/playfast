# playfast
+ Reference
  - https://fastapi.tiangolo.com/ko/ 
+ Environment
  - python 3.11.7
  - fastapi 0.108.0
  - uvicorn 0.25.0

## Install FastAPI
+ install packages
  ```
  conda install fastapi
  conda install uvicorn
  ```
+ add **main.py** and run uvicorn server
  ```
  uvicorn main:app --reload
  ```
